# Calculus
Visualisation For Humans. By Humans.

# Usage
```js
var express = require('express')
var app = express()

app.get('/', function (req, res) {
  res.send('Hello World')
})

app.listen(3000)
```

## Installation

```bash
$ npm install calculus
```
